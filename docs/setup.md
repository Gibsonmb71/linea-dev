# Setup Notes

## Install from GitHub

Clone or download this repository into your WordPress themes folder:

```bash
git clone https://github.com/weeklywildcat/wordpress-theme.git wp-content/themes/weekly-wildcat
```

Then activate **Weekly Wildcat** in WordPress Admin.

## Recommended WordPress settings

- Set permalinks to **Post name**.
- Create categories such as News, Sports, Opinion, Arts, Features, and Announcements.
- Set a navigation menu in the Site Editor.
- Add the school logo through WordPress, not directly in the public repo, unless the logo is approved for redistribution.
